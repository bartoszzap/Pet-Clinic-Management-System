# Pet Clinic Management System
A pet clinic management system developed in Java to fulfill real world scenario features which would be effective in managing pets within a clinic.

# Storage
The information entered gets stored into a text file (PetDetails.txt) allowing for reaccessing data once the program is re-opened. 

The following information gets stored:
- Pet Type
- Pet Name
- Pet Age
- Pet Colour
- Pet Weight
- Pet Breed
- Last Visit

# Features
- Add Pets
- Remove Pets
- Modify Pets
- View All Pets
- View Clinic Report
- Search for a Pet
